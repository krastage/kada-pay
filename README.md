## Installation

```
# Step 1: Clone the repository
git clone https://github.com/krastage/kada-pay.git

# Step 2: Navigate to the project directory
cd kada-pay

# Step 3: Install dependencies
npm install

# Step 4: Start the app && run server
ng serve
```
